Semantic Considerations

This element is expected to be used to signify when navigation items will be listed. Content that is normally wrapped up by the nav element are lists. Not all navigation items need to be contained by the nav element, the footer element often includes links, but for significant blocks of links it should be. List applications include menus, indexes, categories, external links, tables of contents, and more.

Common Attributes and Values

Only global attributes apply to the nav element. Global attributes are shared by all html elements. The a element, or anchor element, is used to link URL addresses.

Accessibility Considerations

To improve accessibility to the nav element, aria-labelledby can be use. According to the MDN website, the aria-labelledby attribute "establishes relationships between objects and their label(s), and its value should be one or more element IDs, which refer to elements that have the text needed for labeling"
