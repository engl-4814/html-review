The "Bottom" of the Webpage

A footer typically contains information about the author of the section, copyright data or links to related documents.

Semantic Considerations

Even though it is typical to see footer elements at the bottom of a page, it does not necessarily need to go there. A footer can go at the top of a page even, a footer holds any extra information you want to add.

Common Attributes and Values

The footer element has global attributes and event attributes.

Accessibility Considerations

If the <footer> element is used as other related information related to the main content, then apply the role="contentinfo" ARIA landmark.
