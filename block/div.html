MDN Definitions

The HTML div element is the generic container for flow content. It has no effect on the the content or layout until styled in some way using CSS.

Semantic Considerations

The <div> element is used to group content. It is styled using class, id, lang, or other attributes.

Common Attributes and Values

The div element includes global elements. Global elements are ones that apply to all HTML elements. This applies to non-standard elements as well. The <div> element is used when other semantic elements are not.

Accessibility Considerations

The <div> element is commonly seen in accessibility through the <button>. To a screen reader, this element will appear as a button control. Through aria, this can create toggle buttons. A button is a widget used to perform various commands on a page. The "Enter" and "Space" keys on the keyboard may activate the button.
