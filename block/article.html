Articles are a way to direct viewers to a separate element within the document or web page.

The HTML article element represents a self-contained composition in a document, page, application, or site, which is intended to be independently distributable or reusable.

Semantic Considerations

An article is a semantic element, like a header or footer, because it accurately describes the content within an element.

Common Attributes and Values

This element only includes the global attributes.

Accessibility Considerations

The ARIA article role is similar to the article element; however the article element should still be given the ARIA role of article, since not all assistive technologies support HTML5 yet.

Example Uses

A given document can have multiple articles in it; for example, on a blog that shows the text of each article one after another as the reader scrolls, each post would be contained in an article element, possibly with one or more sections within.
