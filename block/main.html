The Purpose of Your Webpage

Semantic Considerations

main is where the primary content (the purpose) of the web page goes. Artical, heading, and paragraph tags are typically wrapped in the main tags.

Common Attributes and Values

Accessibility Considerations

Main is typically assign a landmark role. Only one element can be assigned the main landmark role

  • "main" (value)
  • role="main" (element)
