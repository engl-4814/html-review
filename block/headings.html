Headings Signify a New Section

"The HTML h1–h6 elements represent six levels of section headings. h1 is the highest section level and h6 is the lowest."

Semantic Considerations

Headings are the rpimary way to mark different sections of content. Headings can help guide the user to a specific section they want to view, making it easier to search through the web page. There are six differnt types of headings, h1-h6. The higher the number, the less prominent the heading, so the first heading on a page is usually h1.

Common Attributes and Values

All heading elements only include the global attributes.

Accessibility Considerations

All heading elements have the Implicit ARIA role of 'heading'.

All heading elements have the following Permitted ARIA roles;

  • tab
  • presentation
  • none
